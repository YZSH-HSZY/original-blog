# issrc(Inno Setup)

Inno Setup 是一个用于 Windows 应用的开源安装程序制作工具。自 1997 年推出以来，Inno Setup 一直被各种规模的开发者和组织信赖，用于在全球数百万台电脑上可靠地部署软件。
