## 术语

### 零长数组和柔性数组

```c
struct example {
    int count;
    char data[0]; // 零长数组（柔性数组的旧式写法）
};
```

|特性	    |零长数组（旧式）	       |柔性数组（C99 标准）    |
|----------|-------------------------|-----------------------|
|声明	    |char data[0];	          |char data[];          |
|标准支持	|非标准（编译器扩展）	    |C99/C11 标准           |
|内存布局	|与柔性数组相同	           |相同                   |
|使用场景	|动态结构体（可变长数据）	|同左                   |
